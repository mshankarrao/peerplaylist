# PeerPlay list

This is a peerplay list app created with [Angular CLI](https://github.com/angular/angular-cli) version 10.0.4.


## Build and run

Install dependencies by running:

<pre>
npm install
</pre>

Then run a dev server:

<pre>
ng serve
</pre>

To preview, navigate to:

<pre>
http://localhost:4200/
</pre>
